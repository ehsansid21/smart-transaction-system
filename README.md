# Smart Transaction & Fraud Detection System 

## 📌 Project Overview
A backend system that simulates a banking system with smart fraud detection.

Users can:
- Register & login (JWT - upcoming)
- Deposit money
- Withdraw money
- Transfer money
- Detect suspicious transactions

---

## 🛠️ Tech Stack
- Java
- Spring Boot
- MySQL (upcoming)
- Spring Security (JWT - upcoming)
- Spring AI (planned)

---

## ⚙️ Features (Planned)
- Transaction management using @Transactional
- Fraud detection (rule-based + AI)
- Pagination & filtering
- Role-based access (Admin/User)
- Email alerts
